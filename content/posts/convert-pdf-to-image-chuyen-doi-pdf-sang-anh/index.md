EXPVN - Pdf2Image là tiện ích nhỏ, miễn phí, giúp bạn chuyển đổi hàng loạt file PDF sang ảnh với định dạng .JPG
**Hướng dẫn sử dụng:**  
1 - Bạn copy các file PDF cần chuyển sang ảnh vào 1 folder mới  
2 - Mở phần mềm và bấm nút Folder rồi chọn đến thư mục chứa các file PDF (lưu ý: ứng dụng chỉ chuyển file ở thư mục gốc (root) không chuyển file trong các thư mục con)  
3 - Nhập độ phân giải của ảnh vào ô Resolution (lưu ý: nếu bạn chỉ sử dụng ảnh trên web hay gửi mail.... thì bạn có thể để mặc định (72) hoặc chỉnh lên 150. Nếu bạn cần chất lượng ảnh cao hơn (để in) thì nên để độ phân giải 300 hoặc hơn. Kích thước ảnh xuất ra sẽ phụ thuộc vào độ phân giải mà bạn đặt.  
4 - Bấm Convert để tiến hành chuyển đổi. Các file ảnh sẽ nằm trong thư mục tương ứng với tên file pdf.
